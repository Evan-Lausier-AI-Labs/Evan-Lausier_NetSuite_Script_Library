# Evan-Lausier_NetSuite_Script_Library
A comprehensive library of NetSuite SuiteScript solutions

## Structure
Each script is organized in its own folder with a README describing its purpose and usage.

## Scripts

| Script | Type | Description |
|--------|------|-------------|
| [Auto_Email_Invoice](./Auto_Email_Invoice) | Map/Reduce | Automatically email invoices with PDF attachments |
| [Field_Protect](./Field_Protect) | User Event | Field protection script |
| [Forced_Error](./Forced_Error) | User Event | Force error script for testing |
| [Invoice_Grouping](./Invoice_Grouping) | User Event | Group invoices for printing |
| [Email_Script](./Email_Script) | User Event | Email records script |
| [External_ID_Update](./External_ID_Update) | Scheduled | Scripted external ID update |
